🚀 DÉMARRAGE RAPIDE

  PRÉREQUIS : Node.js 18+ (voir INSTALL.md)

  1. Installer Node.js
     → brew install node
     OU télécharger depuis https://nodejs.org

  2. Installer les dépendances
     → npm install

  3. Lancer l'application
     → npm run dev

  4. Ouvrir le navigateur
     → http://localhost:3000

  5. Jouer !
     → Créer une partie et s'amuser

🎯 RÈGLES DU JEU

  1. Créer une partie (2+ joueurs)
  2. Choisir catégories et ordre
  3. À tour de rôle, lire une blague
  4. S'éliminer si on rit
  5. Le dernier en lice gagne !

  • 2 refus max par joueur
  • Pas de répétition de blague
  • Élimination auto-déclarée
  • Statistiques de fin
